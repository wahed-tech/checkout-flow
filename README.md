# Flow Component Demo

## Usage
Use `CardScreen` from `src/CardScreen.tsx`

### Props
| Name                   |             Description              |
|------------------------|:------------------------------------:|
| cardTokenized          | Invoked when card has been tokenized |
| cardTokenizationFailed | Invoked on cart tokenization failed  |
| key                    |             Checkout key             |
